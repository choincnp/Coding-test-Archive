# Array(배열)

##### 특징

1. 선형 자료구조.
2. 크기가 고정.
3. 동일한 타입의 데이터로 구성.
4. index를 가짐.

##### 용도

- 특정 요소를 빠르게 읽어야 할 때. (index를 알고있는 경우)
- 다차원 데이터를 다루는 경우.

##### 시간복잡도

- index를 알고있는 경우 : O(1)
- index를 모르는 경우 : O(n)
- 데이터를 삽입하는 경우 : O(n)
- 데이터를 삭제하는 경우 : O(n)
