# Stack(스택)

##### 특징

1. LIFO(후입선출)
2. 정해진 방향으로만 입력 및 삭제가 가능 (입출구가 하나뿐)
3. `순차접근(sequential access)` 방식.

##### 용도

- 브라우저 뒤로가기
- 실행 취소

##### 시간복잡도

- 데이터 탐색: O(n)
- 데이터를 `삽입` 및 `삭제`하는 경우 : O(1)
